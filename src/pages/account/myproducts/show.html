https://www.ikmultimedia.com/contact-support/sw/


IK Multimedia社ページ( https://www.ikmultimedia.com/userarea/account/ )の、
【登録済みユーザー】 もしくは 【新規ユーザー】 より
IK Multimedia社のログイン、アカウント登録を行ってください。

その後、IK Product Manager にてログイン、
当メールにてお届けいたしましたシリアル・ナンバーを入力いただき
ダウンロード、オーサライズしてお楽しみください。



1. IK Product Manager最新版をダウンロード
( https://www.ikmultimedia.com/pm )

2. IK Product Manager「Manage My Products」画面の「(+) Register Product」にてシリアル・ナンバーを登録

3. IK Product Manager「Software」タブ「Install」、「Authorize」にて購入製品をインストール、オーサライズ

4. SampleTank、AmpliTube、Syntronik、T-RackSなどのシリーズ製品は、ご購入いただいた製品のシリアル・ナンバーがこのタブにまとめて表示されます。


※ Product Managerの詳細につきましては、下記マニュアルをご参照ください。

IK Product Manager User Manual(日本語付き)
( https://www.ikmultimedia.com/products/productmanager/index.php?p=manuals )
